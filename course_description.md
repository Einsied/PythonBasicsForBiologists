# Course description

## Purpose

Python is a [popular](https://github.blog/news-insights/octoverse/octoverse-2024/) and rather easy to programming language.
The continuing digitalization of scientific work turns computer-literacy into a key-competence to achieve personal and career goals in research and industry.
Course participants should acquire fundamental computer-science knowledge as soon as possible,
which is the purpose of this course.

## Goal

Participants will be enabled to create their own short programs and to study Python or other programming languages on their own in the future.

## Description 

The course teaches basic programming in Python on the example of an analysis-task based on neuronal data.
The focus are theoretical foundations, problem-solving strategies and differences between "biological" and "computer-science" approaches to problems.
The course will introduce general concepts and the language itself by creating code for the analysis-task.
The participants and the tutor interactively develop the solution and code to solve the analysis task.

## Examination

The participation in the course and the final result will be used to grade the participants,
if grades are required for the participants programs.
