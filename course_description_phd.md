# Course description

## Purpose
Neuroscience is an interdisciplinary subject integrating behavioral, physiological and molecular techniques. 
Modern experiments create large data-sets requiring advanced and often customized analysis methods.
This course introduces Python as a tool for custom data-analysis,
based on a small sample from a larger data-set.

Python is a [popular](https://github.blog/news-insights/octoverse/octoverse-2024/) and rather easy to programming language,
with active communities developing programs and packages for advanced data-analysis.
To facilitate the adaption of these,
participants will acquire fundamental computer-science knowledge
together with basic knowledge in Python.
This should enable them to enter the begin utilizing more specialized work based on Python.

## Goal

Participants will be enabled to create their own short programs 
and to begin utilizing the work their community already did in Python.
The course will also be ease the acquisition of other programming languages.

## Description 

The course teaches basic programming in Python on the example of an analysis-task based on neuronal data.
The focus are theoretical foundations, problem-solving strategies and differences between "biological" and "computer-science" approaches to problems.
The course will introduce general concepts and the language itself by creating code for the analysis-task.
The participants and the tutor interactively develop the solution and code to solve the analysis task.
